# NFT Project

This project showcases an NFT (Non-Fungible Token) gallery website created using CSS and HTML. Our goal is to allow users to explore different digital assets and help them understand the concept of NFTs.

## `Features`

- **CSS Design:** CSS is used for the visual design of the web page.

- **User-Friendly Interface:** A simple and understandable interface allows users to navigate comfortably.

---

## `Technologies Used`

- **HTML:** The structure of the web page is created using HTML.

- **CSS:** CSS is used for visual styling and layout adjustments.

---

## `How to Use`

1. `Deploy` : Visit the Web Page: Open the [https://nft-project-html-css.netlify.app/](https://nft-project-html-css.netlify.app/) website in your browser.

2. Explore Digital Assets: Browse through different NFTs and digital assets to discover a variety of creations.
